# Integrations

This document tracks external services, APIs, and third-party platforms integrated into the application.

## Frontend Integrations
- **Google Maps**: Basic external link integration `https://maps.google.com/?q=...` (not an embedded API currently, but directs users to Maps).
- **WhatsApp**: Direct chat link generation using `https://wa.me/...` via the floating action button (FAB).
- **Social Media**: Links to standard social platforms (Instagram, Facebook, Twitter, YouTube).

## Planned Backend Integrations
- **Email Delivery**: Will integrate with an SMTP provider (e.g., Gmail with App Password, Resend, Brevo, or SendGrid) to send reservation confirmations to guests and notifications to admins.
- **Database Hosting**: Will likely connect to MongoDB Atlas for cloud database hosting.
