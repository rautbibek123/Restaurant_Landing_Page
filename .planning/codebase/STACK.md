# Tech Stack

This document outlines the core technologies, frameworks, and tools used in the Annapurna Kitchen Restaurant project.

## Frontend
- **Framework**: React 18 (via Vite)
- **Language**: JavaScript (ES6+)
- **Styling**: Vanilla CSS3 (Custom Properties / CSS Variables)
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Build Tool**: Vite

## Backend (Planned)
- **Runtime**: Node.js
- **Framework**: Express.js
- **Database**: MongoDB
- **ODM**: Mongoose
- **Email Service**: Nodemailer (or third-party SMTP like Resend/Brevo)

## Tooling & Quality
- **Package Manager**: npm
- **Linting**: ESLint (with React Hooks and Refresh plugins)
- **Code Formatting**: Prettier (implicit/standard IDE integration)

## Core Dependencies
From `package.json`:
- `framer-motion`: For fluid UI animations, lightboxes, and transitions.
- `lucide-react`: For lightweight SVG icons.
- `react`, `react-dom`: Core UI library.
